Fieldbook & Plot Register Viewer

Developed for the Survey Office for the easy and fast printing of the Survey Fieldbook and Plot Register copies in the Office.

About Fieldbook Viewer

Fieldbook and Plot Register Viewer is the desktop based application developed for the Survey Offices in Nepal to easy viewing and printing of the Fieldbooks and Plot Register in the Offices.

Computer Network

This system is designed to work on the files stored on the computer network.

Employee Friendly UI

It has Employee Friendly UI so the employee can use it easily.

Easy Print

Easily print the copies of the fieldbook and plot register with the your own templates.

Useful Features for Survey Offices

Everything you need to manage your fieldbooks and plot registers efficiently, from indexing to printing.
